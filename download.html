Frequently Asked Questions

Q: Why does my antivirus software mark the Jam54 Launcher as a virus?

The Jam54 Launcher's binaries are often flagged by various antivirus software. Unfortunately, there is nothing that can be done about this issue.

Antivirus software uses heuristics to determine if a program is malware, which often results in false positives. If you trust the developer of the project, you can safely ignore any antivirus warnings. If you do not trust the developer, you should build the binaries yourself or run the software from source.


Q: Why does my browser mark the installer as 'dangerous'?

This happens because the browser’s security tools have detected that this file is not frequently downloaded and may therefore be considered unsafe. You can bypass this warning by selecting the 'Keep anyway' option.


Q: Why does Windows Defender SmartScreen prevent the installer from running?

Whenever you download an application, Windows SmartScreen checks the file against a list of well-known programs.

An application’s reputation increases with the number of downloads. However, if an application hasn’t been downloaded frequently, Windows may prevent it from running and warn you that it could be dangerous. You can bypass this warning by clicking on 'More info' and selecting 'Run anyway'.
