Frequently Asked Questions

Q: Why does my browser mark the installer as 'dangerous'?

This is because the browser’s built-in security tools detected that this file doesn't get downloaded often. And therefore might be unsafe. You can bypass this warning by clicking on 'discard'.


Q: Why does Windows Defender SmartScreen prevent the installer from running?

Whenever you download an application, Windows SmartScreen will check the file that you downloaded against a list of well known programs.

The more often an application is downloaded, the higher its reputation becomes. However, if an application hasn’t been downloaded very often. Windows prevents the application from starting and warns the user that the application may be dangerous. You can bypass this warning by clicking on more info, and selecting 'Run anyway'.
