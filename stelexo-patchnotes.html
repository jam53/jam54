Stelexo Patch Notes


0.3.0

General

  • Added a procedurally generated, infinite world
  • Added a pause menu
  • The player's position and settings in the secret level are now being saved
  • Added an easteregg to the main menu
  • Reduced the size of the game by 55%
  • The FPS counter that was present on the main menu, has been added in game as well
  • Bugs/crashes can now be reported to the developer using the pause menu
  • Reduced the refresh rate of the FPS counter

Bug Fixes

  • Fixed some issues with the installer
  • Fixed an issue with the anti-aliasing toggle
  • Fixed an issue with the FPS toggle


0.2.0

General

  • Improved player movement
  • The player can hold jump now
  • Added the ability to crouch
  • Added the ability to aim
  • Added camera breathing
  • The Fov changes depending on the speed of the character
  • Added a main menu
    • Adjustable keybinds
    • Adjustable settings
    • A shop
    • Background music
  • Removed the little settings window on startup
  • From now on the game comes with a launcher. This will automatically update the game, eliminating the need to manually download a new update every time there is a new version
  • Added an FPS counter

Bug Fixes

  • Fixed an issue where the player would respawn, when they fell in the water
  • Fixed some issues with gun clipping
  • Fixed a bug that caused some of the debris to be destroyed incorrectly


0.1.3

General

  • Changed the version numbering system of the game (X.Y.Z)
    • The X value will be changed to 1, when the game is fully released. After which it will change when incompatible API changes are made
    • The Y value changes when new functionality is added
    • The Z value changes when bug fixes/small improvements are made
  • Removed the fire inside the volcano, added small flames instead
  • The Quality of the welcome sign was improved
  • Added a different type of smoke to the volcano
  • Shadows are enabled again
  • Changed the color of destroyed trees
  • If the player falls of the map or into the volcano, they will respawn
  • Removed the easteregg
  • Pieces of destroyed objects, now fall through the ground
  • Changed the colliders of destroyed objects
  • Reduced the height of the snow mountain
  • Changed the quality settings

Bug Fixes

  • Fixed an issue where the player could walk through/under the welcome sign
  • Fixed an issue that caused the welcome to be baked incorrectly
  • Removed a very bright light inside the volcano
  • Removed some weird light spots on the sand
  • Patched up some spots where the player could fall through the map
  • Fixed an issue where the player was able to fire by pressing Ctrl
  • Fixed an issue that some objects would float in the air
  • Fixed an issue where the sound of the waterfall would stop playing
  • Fixed an issue where the gunfire sound wouldn't be played on the player
